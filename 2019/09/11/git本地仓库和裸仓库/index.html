git本地仓库和裸仓库

Posted on 2019-09-11
Words count in article: 770 | Reading time ≈ 3

¶git 初始化

git init这个命令执行后会在本地生成一个 .git 的文件夹，用来追踪仓库的所有变更。
--bare: 使用 --bare 参数初始化的仓库，我们一般称之为裸仓库， 因为这样创建的仓库并不包含 工作区 ， 也就是说，我们并不能在这个目录下执行我们一般使用的 Git 命令。
